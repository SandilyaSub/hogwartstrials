# Project Memory

## Core
React + HTML5 Canvas. Strictly 2D platformer. No 3D engines.
Hogwarts Trials © 2026. 7 worlds, 10 levels each, boss at end.
Cloud-synced profiles. Email/password auth. Cloud DB synced.
Fredoka & Quicksand fonts. Pixel-art in-game, AI portraits for menus.
CSS variables dynamically override global UI colors.
Card selections auto-advance on click (no confirmation buttons).
No audio/music in the game. All song features removed.
